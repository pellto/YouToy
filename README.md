# YouToy

유튜브의 기능들을 살펴보고 해당 기능들을 구현하는 백엔드 클론 코딩 프로젝트 

## 프로젝트 목표
- 유튜브의 다양한 기능들을 사용할 수 있어야 한다.
- JDBC -> JPA로 코드 변환을 시도한다.
- 모든 서비스(또는 유스케이스)에 대해 테스트 코드를 작성한다.
- traffic 병목이 일어나는 곳을 테스트하고 개선한다.
- CI / CD를 적용한다.
